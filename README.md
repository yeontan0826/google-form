# 실행 방법

## Step 1: 패키지 설치

프로젝트 폴더로 들어가 `yarn install` 명령어를 통해 패키지들을 설치합니다.

## Step 2: expo 실행

`yarn start` 명령어를 통해 expo를 실행합니다.

## TroubleShooting

**iOS, AOS 에뮬레이터가 작동되지 않을 경우, 개인 휴대폰을 통해 확인이 가능합니다.**
